# 基于Flask的图书管理系统 python1222

## 项目概述

本项目是一款基于Flask框架和MySQL数据库的图书管理系统，使用Python进行开发。系统支持管理员和用户两种角色登录，并具备完善的注册功能。

## 技术栈

- 前端：HTML, CSS, JavaScript, Bootstrap
- 后端：Flask
- 数据库：MySQL
- 开发语言：Python

## 功能模块

### 基础功能

- 图书信息的增删改查
- 用户注册与登录
- 管理员登录

### 扩展功能

- 用户评论

## 系统角色

- 管理员：管理图书信息，查看用户评论
- 用户：检索图书信息，发表评论，进行注册和登录

## 运行环境

- Python 3.x
- Flask 1.x
- MySQL 5.x

## 部署步骤

1. 安装Python环境
2. 安装Flask框架
3. 安装MySQL数据库
4. 导入数据库结构
5. 运行Flask应用程序
